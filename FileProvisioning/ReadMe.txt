Open the solution in Microsoft Visual Studio 2015.  
Restore the referenced libraries from their packages at Nuget.org, by doing as follows: 

1.  Open the Package Manager Console by choosing Tools|Nuget Package Manager|Package Manager Console from the menu.  
2.  In the Package Manager Console, choose nuget.org from among the options in the Package Source drop-down list.  
3.  Execute this command from the Package Manager Console prompt: 
    
    Update-Package -Reinstall
